# ZEPTo

===========================================
           ZEPTO INVENTORY DATASET
===========================================

Overview
--------
The Zepto Inventory Dataset contains product-level information collected from
Zepto, one of India's leading quick-commerce platforms. It includes details
such as product names, categories, pricing, discounts, stock availability,
and packaging information. This dataset is suitable for data analysis,
business intelligence, SQL practice, and machine learning projects.

Dataset Contents
----------------
The dataset includes information such as:
- Product ID
- Product Name
- Category
- Brand
- Maximum Retail Price (MRP)
- Selling Price
- Discount Percentage
- Available Quantity
- Weight / Volume
- Stock Availability

(Note: Column names may vary depending on the dataset version.)
